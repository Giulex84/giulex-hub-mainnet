Privacy Policy

Last updated: 2026

Arena is a skill-based memory game built on Pi Network.

1. Data Collection

Arena does not collect, store, or process personal data. Authentication is handled exclusively through the official Pi Network SDK.

When you log in, we receive your Pi username and a unique user identifier (UID) from Pi Network. This information is used only to enable gameplay and payment functionality within the app.

2. Payments

Premium features are unlocked via optional Pi payments. All payment processing is handled by Pi Network. Arena does not store wallet information or private keys.

3. Data Storage

Arena does not store personal user data on external databases.

4. Third-Party Services

Arena relies solely on Pi Network for authentication and payment processing. No additional third-party services are used.

5. Changes

This Privacy Policy may be updated if the app functionality changes.
